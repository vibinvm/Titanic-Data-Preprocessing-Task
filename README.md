# Titanic Data Cleaning & Preprocessing

## Objective
Clean and prepare Titanic dataset for Machine Learning.

## Steps Performed
- Handled missing values (Age, Embarked, Cabin)
- Encoded categorical variables
- Standardized numerical features
- Detected and removed outliers using IQR
- Visualized data using boxplots

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Dataset
Titanic Dataset (Kaggle)

## Outcome
Prepared clean dataset ready for ML model training.
